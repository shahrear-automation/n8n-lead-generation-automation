# 🚀 AI Lead Generation Automation System (n8n)

An advanced automation workflow built with **n8n**, **Gemini AI**, and **Google Maps scraping** to generate leads and deliver structured reports automatically.

![Workflow Architecture](./workflow-architecture.png)

---

## 📌 Overview

This system automates lead generation from user requests via Telegram or Email, processes queries using AI, fetches data from Google Maps, and generates structured PDF reports.

---

## ⚙️ Key Features

✅ Multi-channel input (Telegram + Email)  
✅ AI-powered query understanding (Gemini)  
✅ Automated Google Maps data scraping  
✅ Structured lead report generation  
✅ PDF export system  
✅ Telegram delivery of reports  
✅ Smart routing for different response types  

---

## 🧠 How It Works

1. User sends request via Telegram or Email  
2. Gemini AI processes and understands intent  
3. System builds a search query  
4. Google Maps scraper fetches business data  
5. Data is structured into a report  
6. PDF is generated automatically  
7. Report is delivered via Telegram  

---

## 🛠 Requirements

- n8n
- Telegram Bot API
- Gmail API (optional)
- Google Maps scraping (Apify or custom)
- Gemini API (LLM)

---

## 🚀 Setup

1. Import `.json` workflow into n8n  
2. Configure credentials  
3. Connect Telegram bot  
4. Set API keys  
5. Activate workflow
